# Privacy Review

Use this checklist before publishing:

- No credentials, tokens, API keys, private keys, or wallet data.
- No private Codex Desktop state or conversation history.
- No local absolute paths.
- No private logs or runtime databases.
- No private repository history.
- No internal hostnames or service-only configuration.
- Examples use placeholder values only.

Recommended placeholders:

- `example.com`
- `user@example.com`
- `REPLACE_ME`
- `YOUR_TOKEN_HERE`
- `sample-project`
