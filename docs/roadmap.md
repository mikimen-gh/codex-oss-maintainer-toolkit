# Roadmap

## 0.1

- Provide a dependency-free maintainer readiness report.
- Detect common secret patterns and private local-state markers.
- Publish Markdown and JSON output for automation.
- Run the report in CI for every push and pull request.

## 0.2

- Add issue and pull request template checks.
- Add release checklist validation.
- Add SARIF output for security tooling.

## Later

- Integrate optional GitHub API metadata.
- Track maintainer workload signals over time.
- Generate release notes from sanitized public repository data.
