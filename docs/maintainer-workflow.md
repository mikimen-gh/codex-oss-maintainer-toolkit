# Maintainer Workflow

## Review

- Check that the change has a clear purpose.
- Confirm tests or documentation were updated when needed.
- Run the public-safety check.

## Release

- Confirm the changelog is updated.
- Run tests.
- Verify that examples still use placeholder values.
- Tag only after the repository is clean.

## Codex Usage

Codex can help with review, test generation, documentation updates, release
notes, and privacy checks. Final publication decisions remain with the
maintainer.
